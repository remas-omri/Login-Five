# Login Five – Android App

A simple Android application developed using **Android Studio** as part of a course project.  
The app provides basic user authentication features including **user registration**, **login**, and a **welcome screen**.

---

## 📱 Features
- Create a new user account  
- Login with username and password  
- Password confirmation during registration  
- Clean and simple UI screens  
- Smooth navigation between app pages  

---

## 🛠️ Technologies Used
- Android Studio  
- Java  
- XML Layouts  
- Basic Activity Navigation
